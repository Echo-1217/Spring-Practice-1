### Restful API

#### 4.Restful具有哪些特性或優點?
:::warning
1. 使用者介面與資料分離
2. 可以直接預期Request會收到的資料類型
3. 可以更高效地利用 cache 來達到更快的回應速度
4. 無狀態的特性，通過URI對資源操作時，沒有上下文的影響
5. 統一介面簡化系統架構，減少耦合性
:::
